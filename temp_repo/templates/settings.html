Help & Information
Creating OKX API Keys
  1. Log in to your OKX account
  2. Go to "API Management" in account settings
  3. Click "Create API Key"
  4. Set permissions to "Read" and "Trade"
  5. Copy API Key, Secret Key, and set a Passphrase
Trading Safety
  • Always start with Paper Trading mode
  • For Live Trading, set strict API access permissions
  • Never share your API credentials with anyone
Setting Up Telegram Notifications
  1. Message @BotFather on Telegram to create a bot
  2. Follow instructions to create a new bot
  3. Copy the provided token
  4. Message @userinfobot to get your Chat ID
